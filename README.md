# LearningRacers_Release

UnityとUnity ML-Agentsを用いたAIカーレースゲームです。

## 環境

- Unity 2022.3.58f1
- Unity ML-Agents Release 19
- Blender 4.2.2（モデリング一部使用）

## 内容物

- `/Game`：ビルド済みアプリ
- `/Script`：主要スクリプト（プレイヤー・AI・UI など）
